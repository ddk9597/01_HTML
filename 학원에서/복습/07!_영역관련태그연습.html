style 태그 내용 적용 확인

block 형식

        - 화면(공간)을 수직 분할 : 세로 화면을 가로로 잘라서 층을 나눔

        - 한층을 모두 차지함

        - width, height와 같은 크기 지정 속성 사용 가능

        - block 형식의 태그
        h1~h6(제목)     p,pre(문단)
        hr(수평선)      div(영역 분할)
        시맨틱 태그(div 태그에 이름 넣은 태그)

h3는 block 형식 태그입니다

        block 형식은 작성된 내용이 짧더라도
        한 줄을 모두 차지하려는 성질 때문에
        다음 태그 내용은 다음 줄에 작성됩니다
첫 번째 div
두 번째 div
세 번째 div

inline 형식

        - 화면을 수평 분할 : 가로 화면을 세로로 잘라서 방을 나눈다

        - 한 층을 모두 차지하지 않음 : 내부에 작성된 내용 크기만큼 공간 차지하려는
        
        - 별도로 크기 지정 속성 사용 불가

        - inline 형식의 태그
        b(굵게)                 i(기울이기)
        strong(접근성 강조)     em(접근성 기울이기)
        mark(형광펜)            span(영역분할)  등
inline 형식은행을 분할하는 성질을 가지고 있어 줄이 바뀌지 않고 계속 옆으로 작성된다
첫 번째 span 두 번째 span 세 번째 span

영역 지정 방식의 차이점

  • block 형식 : 직사각형 박스 형태로 영역을 지정
  • inline 형식 : 내부에 작성된 내용 단위로 영역 지정

inline-block 형식

        - inline 의 특징 중 수평 분할(세로로 자르는 열 나눔)
        - block의 특징 중 크기 지정 기능이 있는 방식
        -대표적인 태그 : img, *iframe 태그
