img 태그

        - 웹 문서에서 사진이나 그림 같은 이미지를 삽입하는 용도의 태그
        [속성]
        (속성 : 시작 태그에 작성하는 내용)

        1. src : 삽입할 이미지의 경로를 작성하는 속성
                (파일위치, 인터넷 주소)

        2. width/height : 너비, 높이를 지정하는 속성(이미지의 크기 지정)
        
        3. alt(대안) : 이미지 설명을 작성하는 속성
                    ->이미지가 출력되지 않은 경우 화면에 출력됨
                    + 웹 접근성(스크린 리더가 alt에 작성된 이미지 설명을 읽어줌)

img 예시

절대 경로로 이미지 출력하기

        절대경로: 변하지 않는 기준점으로부터 목표까지의 경로
         (ex : 서울시 중구 남대문로 120 청계 그레이츠 2층)
        - 윈도우 기준 : C드라이브, D드라이브 기준

상대경로로 이미지 출력하기

 
        상대경로 : 현재 파일/페이지 위치를 기준으로 하여 
                목표까지의 경로를 나타내는 것 (라이브 서버에서 보인다)

고정 크기 단위 (px) : 항상 고정된 크기를 지닌다

가변 크기 단위(%) : 화면 또는 영역에 대한 상대적인 크기 단위

고정 + 가변 크기

        고정 크기로 설정된 요소(태그) 내부에서 
        일정한 크기의 이미지 또는 요소를 출력하고 싶을 때
        가변크기로 지정하면 된다.
