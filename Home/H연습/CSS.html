일반 구조(위치) 선택자

        html, css의 위치를 기준으로 선택
        형제 관계에 있는 요소 중에서 특정 요소만을 선택하는 선택자

        *위치를 기준으로 구분함*

        ( A: 선택된 요소)

        A: first-child -> 형제들 중 첫째가  A가 맞다면 선택
        A: last-child -> 형제들 중 막내가  A가 맞다면 선택
        A: nth-child(a)-> 형제들 중 a번째가가  A가 맞다면 선택
        A: nth-last-child(a)-> 형제들 중 마지막에서 a번째가가  A가 맞다면 선택

        child(요소구분안함), type(요소 구분함)

부정 선택자

        A:not(B) 
        선택된 모든 A 중 B만 빼라
    
        CSS
        #test3>li:not(:nth-of-type(3n)){ 
            background-color: pink;
        }

only-of-type

        특정 요소의 자식 중 지정된 자식 요소가 딱 하나만 있을 때 선택

        

            
자식1

            
자식2

        


        /* #test3의 자식 p 태그를 모두 선택했는데 딱 하나만 존재할 경우 */

        #test3 > p:only-of-type{
            background-color: pink;
        }

:empty

자식노드(자식요소)가 없는 노드(요소)인 경우 선택
노드란 HTML에 작성되어지는 모든 것들
ex) 태그, 글자, 띄어쓰기, 엔터 주석 등 모두 다
**시작, 종료 태그 사이에 아무것도 존재해서는 안됨**
