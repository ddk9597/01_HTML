form tag

입력 양식을 작성하는 판

내부에 작성 된 입력 값을 지정된 곳으로 제출하는 역할

제출 주소 작성

input

input type="text"
input type="password"
input type number
input type range 
        input 속성
        min, max
        step : 증가, 감소하는 간격
        value : 초기값 지정

날짜, 시간 관련 input 태그

radio, check

        여러 연관 값을 묶어서 선택하는 경우에 사용
        radio : 하나만 선택
        checkbox : 여러개 선택
        연관된 radio, checkbox는 같은 name 속성 값을 가져야 한다

        radio, checkbox에서 제출 할 값을 수신하는 곳에서 표기 방식을 작성하는 속성

label 태그

        입력 관련 태그 등의 제목을 나타내는 태그
        1. label 태그 내부에 입력 관련 태그를 하나 작성하면 
        label 태그를 클릭해도 입력 관련 태그가 선택된다

        2. 입력 관련 태그의 id 속성 값을 label 태그의 for 속성 값과 동일하게 설정하면
        원거리에서 연결 가능하다
