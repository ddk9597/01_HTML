form 태그 / input 태그

form 태그

        - 입력 양식을 작성하는 판(영역)
        - 내부에 작성 된 입력 값을 지정된 곳으로 제출하는 역할

        [속성]
        1) * action : 제출될 곳을 지정하는 속성(주소를 작성)    
        2) * method(방식) : 설정된 곳으로 데이터를 전달하는 방식을 설정
        3) name : 이름 지정하는 속성(JS사용)
        4) target : 제출된 페이지를 현재/새 탭으로 열기

input 태그

        - 웹 문서에서 사용자가 입력 할 수 있는 양식을 제공하는 태그

        - 많은 type이 존재하고, input의 변형 형태도 다수 존재하고
        
        * input 태그에는 name 속성이 필수로 작성 되어야 한다!
        (꼭은 아니지만 작성하는 경우가 많음( 80% 이상 ))

숫자 관련 input 태그

        type="number" : 숫자만 입력 가능한 입력 상자

        type="range" : 슬라이드바를 이용해서 숫자를 지정
                        (단독사용되는 경우는 별로 없고 JavaScript와 혼용)
        [속성]
        1. min : 최소값 (입력 범위 정하기)
        2. max : 최대값 (입력 범위 정하기)
        3. step : 숫자가 증가 / 감소하는 간격(단계)
        4. value : 초기값(기본값) 지정
